**Radio** is a plugin for the AshlynSMP server.

## Description

Radio is a comprehensive Spigot/Paper plugin designed specifically for the AshlynSMP server. It manages all radio-based features for the server.

## Building from Source

### Prerequisites
- Java 21 or higher
- Maven 3.6+

### Build Steps

```bash
# Clone the repository
git clone https://github.com/AshlynSMP/Radio.git
cd Radio

# Build with Maven
mvn clean package

# The compiled jar will be in the target/ directory
```

## Requirements

- **Minecraft Version**: 1.21+
- **Server Software**: Spigot/Paper 1.21+
- **Java Version**: 21
